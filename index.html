Why Shapes?

The only theme with a native comparison table — Phab’s critical conversion feature

Shapes is the only theme with a built-in “Us vs Them” comparison table — the critical feature for converting first-time protein buyers (as seen on the Alpino reference site). Combined with 100% positive reviews, food-industry DNA, and playful design matching Phab’s brand personality, it provides the strongest foundation for the revamp.

What Shapes Gives You (Free)

  • ✓  Comparison table (unique to Shapes)
  • ✓  Nutritional info sections
  • ✓  Sliding flavor scales
  • ✓  Playful animations & dividers
  • ✓  Industry icons for consumables
  • ✓  All 13 CRO capabilities covered

What Needs Customization

  • ⚙  Sticky ATC (2–4 hours custom code)
  • ⚙  Phab brand colors & typography
  • ⚙  “Products Sold” counter (like Alpino)
  • ⚙  Automation flows (abandoned cart)
  • ⚙  GA4 setup via Google/YouTube app
